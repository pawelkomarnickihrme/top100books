Welcome to Hikari, my open source project for building a SaaS application with Next.js.

This project is a Next.js starter kit that is designed to help you get up and running with your next SaaS project. Special thanks to the following projects for making this possible:

- [Supabase](https://github.com/supabase/supabase)
- [Stripe](https://github.com/stripe)
- [Vercel](https://github.com/vercel/vercel)
- [MagicUI](https://github.com/magicuidesign/magicui)
- [SyntaxUI](https://github.com/SyntaxUI/syntaxui)
- [Fumadocs](https://github.com/fuma-nama/fumadocs)
